Assorted Additions

What is it about?

Assorted Additions is a "medium" sized content mod for the game Terraria that adds all sorts of things to the base game like the name implies. In its current state the mod adds a little over 200 new items (from basic blocks to more complex weapons) for the player to discover. The mod also adds 15 new enemies and two bosses to the game along with a bunch of tweaks. For example, the mod adds a new category of "accessory" to the game called runes and improves the loot the player can find in underground mini biome chests.

Some background

I was doing a modded playthrough of Terraria with some friends and at some point we were discussing the question "How hard could making a mod be" probably inspired by some bug we had found. A few moments later my friend had jokingly modded in an artillery cannon, albeit a very scuffed one. After this I also decided to give modding a go.

It turns out that getting started on modding is very easy even with zero knowledge of C# or modding in general. To be fair, I did have some experience in Java through my studies so the C# syntax was easy to understand and I had general knowledge on programming concepts. On top of this tModLoader provides plenty of examples and documentation to get things started. In no time I had added a basic steel sword to the game. After that I kept adding basic items as new ideas came up.

It did take a ton of time and effort to start actually making things that were decent. On top of having to learn the modding API I had to learn how to make sprites. I did struggle a lot with making sprites and still do. Coming up with a visual idea and actually implementing it is very time consuming. To answer the question of "how hard could making a mod be" I would say that it depends. It is as hard and complex as the thing you are trying to implement. A basic sword can be added in minutes, an unique and complex boss can take weeks.
